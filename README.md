# 📌 Lost & Found Intelligence System

A web-based Lost & Found Intelligence System that helps users report lost items and found items, making it easier to reconnect owners with their belongings.

This system allows users to register, login, report items, and search for lost/found items in a simple and efficient way.



# 🚀 Features

✅ User Registration & Login
✅ Forgot Password with OTP verification
✅ Report Lost Items
✅ Report Found Items
✅ Search & View Items
✅ Secure Authentication
✅ Responsive UI



# 🛠️ Tech Stack

# Frontend

* React.js
* Tailwind CSS
* JavaScript
* React Router

# Backend

* Node.js
* Express.js

# Database

* MongoDB
* Mongoose

# Other Tools

* Nodemailer (for OTP email)
* JWT Authentication



# ⚙️ Installation & Setup

# 1️⃣ Clone the repository

git clone https://github.com/your-username/lost-found-system.git




# 2️⃣ Install dependencies

Backend:

bash
cd server
npm install


Frontend:

bash
cd client
npm install




# 3️⃣ Setup Environment Variables

Create `.env` file in server folder:


PORT=5000
MONGO_URI=your_mongodb_connection
EMAIL=your_email
EMAIL_PASS=your_email_password
JWT_SECRET=your_secret_key




# 4️⃣ Run the project

Backend:

bash
npm start


Frontend:

bash
npm run dev




# 🔐 Authentication Flow

1. User registers
2. User logs in
3. Forgot password → OTP sent to email
4. OTP verification
5. Reset password
